# Weather Dashboard

A weather dashboard built with **Node.js, Express, EJS, and the OpenWeather API** that displays real-time weather information and allows users to search weather for any city.

---

## Features

- Display current weather for a default location (Toronto)
- Search weather by city name
- Temperature and feels-like temperature
- Wind speed and wind direction
- Humidity level
- Cloud coverage
- Atmospheric pressure
- Error handling for invalid city names
- Clean and responsive weather dashboard UI

---

## Technologies Used

- Node.js
- Express.js
- EJS (Embedded JavaScript Templates)
- Axios
- OpenWeather API
- HTML5 & CSS3

## How the Application Works

1. The server fetches default weather data using the OpenWeather API.
2. The data is passed from Express to the EJS template.
3. Weather information is dynamically rendered on the dashboard.
4. Users can search any city using the search form.
5. If an invalid city is entered, a friendly error message is displayed.
