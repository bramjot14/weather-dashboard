# Weather Dashboard

A weather dashboard built with Node.js, Express, EJS and the OpenWeather API.

## Features
- Display current weather for a location
- Temperature and feels-like temperature
- Wind speed and direction
- Humidity and cloud coverage
- Atmospheric pressure
- Clean weather dashboard UI

## Technologies
- Node.js
- Express.js
- EJS
- Axios
- OpenWeather API

## Run the Project

1. Clone repository
2. Install dependencies

npm install

3. Start server

node index.js

4. Open browser

http://localhost:3000
